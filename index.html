📧 How to Add to Outlook

  1. Click "📋 Copy Signature to Clipboard" button above
  2. Open Outlook → View → View Settings → Accounts → Signatures
  3. Click "New" to create a signature
  4. Enter the name for the new signature
  5. Paste (Ctrl+V) the signature into the edit box
  6. Click Save
  7. For fields "New Messages" and "Replies/Forwards" select your signature name
  8. Click Save
  9. Congratulations, you have added your new signature! 🎉
